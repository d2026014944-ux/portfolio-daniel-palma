## 🏛️ Contexto: Projeto OPD

O [Observatório do Pico dos Dias](https://www.lna.br/opd), em Brazópolis/MG, abriga um telescópio de alta tecnologia fruto de cooperação internacional:

```
┌─────────────────────────────────────────────────────┐
│  Roscosmos (Rússia) + AEB (Brasil) + LNA (Itajubá) │
│  ─────────────────────────────────────────────────  │
│  • Rastreamento de detritos espaciais e satélites   │
│  • Dados massivos em tempo real                     │
│  • Catalogação automatizada                         │
│  • Processamento de imagens noturnas                │
└─────────────────────────────────────────────────────┘
```

Como estudante de Ciência de Dados na UNIFEI (sede do LNA), possuo perfil altamente valioso para contribuição neste projeto.
